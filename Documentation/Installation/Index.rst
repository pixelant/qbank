============
Installation
============


1. Install the extension using Composer: `composer req pixelant/qbank`

2. Activate the extension in TYPO3 by using the *Admin Tools > Extensions* module
   or by running `vendor/bin/typo3 extension:activate qbank; vendor/bin/typo3cms database:updateschema` in the command line.
